Постановка задачи # soz_project
В ходе анализа лояльности пользователей Яндекс Афишы необходимо определить какие пользователи с большей вероятностью возвращаются на платформу и делают заказы повторно. Для этого необходимо осуществить действия, условно разделенные на следующие этапы:
1. Загрузка данных и знакомство с ними.
2. Предобработка загруженных данных.
3. Создание профиля пользователя.
4. Исследовательский анализ данных:
- исследование признаков первого заказа и их связи с возвращением на платформу;
- исследование поведения пользователей через показатели выручки и состава заказа;
- исследование временных характеристик первого заказа и их влияние на повторные покупки;
- корреляционный анализ количества покупок и признаков пользователя.
5. Общие выводы и рекомендации.
